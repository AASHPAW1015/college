Blockchain-Based Voting System

This project explores the use of blockchain technology to create a secure, transparent, and tamper-resistant voting system. The system is designed to ensure voter anonymity while maintaining verifiable election results through immutable blockchain records.


Each vote is recorded as a transaction, preventing duplication or manipulation. The project focuses on understanding decentralization, consensus mechanisms, and trustless verification.


This project was developed as part of my Class 12 final-year project, with the goal of studying how blockchain can be applied beyond cryptocurrencies, particularly in governance and democratic processes.
